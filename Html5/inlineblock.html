块级元素的特征：

1. 默认独占一行

2. 没有宽度，默认铺满一行

3. 支持所有CSS类

块级元素：

Normal: address blockquote br dd div dl fieldset form h1 h2 h3 h4 h5 h6 hr li main nav noscript ol p pre table tfoot ul

HTML5: article article canvas figcaption figure footer header output section video

See: 块级元素

内联元素特征

1. 同行可以又多个同类标签

2. 宽度由内容决定

3. 不支持width与height

4. 不支持上下margin

5. 代码换行被解析：同类标签位于多行时，标签之间会插入空格

内联元素：

a b big i small tt abbr acronym cite code dfn em kbd strong samp time var bdo br img object q script span sub sup button input label select textarea

See: 内联元素
